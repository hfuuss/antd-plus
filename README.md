# antd-plus
基于antd的业务组件库

https://plus.getshell.cn/
# 如何使用

```bash
$ npm i @@hfuuss/antd-plus -D
```

```js
import React from 'react';
import { Button } from '@hfuuss/antd-plus';
export default () => <Button/>;
```

# 如何开发

安装依赖,

```bash
$ npm i
```

开启服务,

```bash
$ npm start
```

构建文档,

```bash
$ npm run docs:build
```

通过 `father-build` 构建组件库,

```bash
$ npm run build
```
