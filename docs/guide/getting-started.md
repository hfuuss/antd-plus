## 环境准备

首先得有 [node](https://nodejs.org/en/)，并确保 node 版本是 10.13 或以上。

```bash
$ node -v
v10.13.0
```

## 安装

```bash
$ npm i @@hfuuss/antd-plus -D
```

## 使用


```jsx
import React from 'react';
import { Button } from '@hfuuss/antd-plus';
export default () => <Button/>;
```
